# Trademark & Brand Usage

"HXML", project marks, and logos are used to identify the official project.

## Allowed

- Referencing HXML in compatibility statements.
- Linking to this repository.
- Describing unmodified upstream releases as HXML.

## Not Allowed (without permission)

- Using HXML name/logo to imply official endorsement.
- Shipping modified forks under confusingly similar branding.
- Reusing official logos in a way that causes source confusion.

## Forks

Forks are welcome, but should use distinct branding when distributed publicly.

## Contact

For permission requests, use the licensing contact path in `commercial-license.md`.
