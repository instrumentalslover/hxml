# HXML Examples

- `basic-page.hxml` — Simple HTML page with one XML-mode section.
- `mixed-modes.hxml` — Interleaved HTML and namespaced XML-mode content.

Try:

```sh
hxml build examples/basic-page.hxml -o examples/basic-page.html
hxml build examples/mixed-modes.hxml -o examples/mixed-modes.html
```
